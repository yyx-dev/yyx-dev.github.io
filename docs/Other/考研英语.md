# 考研英语

## 阅读方法论

| 阅读技巧                                                     |
| ------------------------------------------------------------ |
| **先题后文，只看题干**<br/>串联题干间逻辑，推测文章主题      |
| **抓作者态度**：感情色彩词 .adj .adv<br/>**抓句间逻辑**：but表转折，没but表逻辑连贯，用来强调重点 |
| **干扰选项特征**<br/>• 正反混淆<br/>• 概念偷换<br/>• 答非所问<br/>• 不同内容嫁接<br/>• 非最佳答案<br/>• 绝对化用词<br/>**正确选项特征**<br/>• 同义改写<br/>• 符合中心思想（细节服从主旨） |
| **标点符号的运用**<br/>• 两个逗号之间为插入语，一个逗号之后为补充语，非主干。<br/>• 冒号前后，一般从抽象到具体，解释前面的内容。<br/>• 分号前后，为并列关系。 |

| 六大题型 | 内容                                                         |
| :------: | ------------------------------------------------------------ |
|  细节题  | 标志：询问某事的具体细节或原因<br/>要求：① 集齐全部定位信息，判断最终答案<br/>&emsp;&emsp;&emsp;② 细节服从中心主旨 |
|  例证题  | 标志：example case illustrate demonstrate to show to sth.<br/>例子本身不重要，重要的是例子所支持的观点；观点一般在例子前，有时也在例子后。<br/>干扰选项：就事论事，只谈例子本身，不谈观点 |
|  主旨题  | 标志：mainly about / discuss / the best title<br/>方法：<br/><mono>1)</mono> 串线法：将各段首末句串联成一个整体，注意转折处。<br/><mono>2)</mono> 中心句：<br/>&emsp;&emsp;形式上：① 如果文章以提问开头，对它的回答，就是文章的中心<br />&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;② 独句段<br/>&emsp;&emsp;内容上：① 时间对比 now<br/>&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;② “少数派原则”，作者观点通常和大众相反 few unrecognized unmention<br/><mono>3)</mono> 中心词：重复提到、高频出现的重点词 |
|  态度题  | 标志：attitude deem consider tone<br/>方法：选项词汇归纳<br/>① 正面：positive optimistic approval supportive<br/>② 负面：negative pessimistic disapproval critical doubtful suspicious skeptical <br/>questionable<br/>③ 中立：neutral objective impartial disinterested<br/>错误选项态度：<br/>uninterested unconcerned indifferent<br/>confused puzzled<br/>biased prejudice subjective cocentemptuous<br/><hr style="margin: 4px 0px;" />作者态度没有明确提出时，怎么找：<br/>① 文章的框架（主旨串线法）<br/>② adj adv（作者情感）<br/>③ 转折处<br/>④ 情态动词之后 should must could |
|  猜词题  | 方法：通过定位好的句子，根据逻辑关系，找同义词或反义词<br/>干扰选项特征：字面意思理解 |
|  推理题  | 标志：infer / learn from / imply / suggest / conclude<br/>方法：① 同义改写——细节／主旨／逻辑<br/>&emsp;&emsp;&emsp;② 逻辑取反——推理多在转折处 |
