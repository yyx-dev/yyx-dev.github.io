## 小建议

1. **不要超过4种背景色**，否则文档像彩虹，反而分散注意力
2. **字体色用深色系**（#1a1a1a、#0d47a1、#2c3e50），浅色字伤眼
3. **统一语义**：黄色=重点，蓝色=术语，红色=易错，绿色=补充
4. **背景色够浅**才能长时间阅读不累
5. **使用技巧**：背景色越浅越护眼，`#ffff00` 这种纯亮黄比较刺眼，建议用 `#fffacd` 或 `#fff3e0` 这类柔和色
